Troubleshooting

iCloud sync issues

  • Make sure the device is signed into iCloud.
  • Open Grocer Settings and confirm iCloud status is available.
  • Pull to refresh after reconnecting to the network.
  • If a shared list became read-only, ask the owner to check your permissions.

Photo import issues

  • Confirm camera or photo library access is allowed in iPhone Settings.
  • Use a clear, well-lit image of the list.
  • Review detected items before adding them to a list.
